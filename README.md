## MambaFNet: Bidirectional Enhancement and Collaborative Aggregation for Small Object Detection in Remote Sensing Images
# The code and experimental results of MambaFNet will be made publicly available here.
Abstract：Remote sensing images are characterized by substantial variation in object scales and complex scene compositions, which often cause existing real-time small object detection algorithms to struggle with modeling long-range dependencies and capturing global contextual information when processing remote sensing targets. To address these challenges, we propose MambaFNet, a small object detection algorithm for remote sensing images based on bidirectional enhancement and collaborative aggregation. Specifically, we construct the Mamba-Enhanced Bidirectional Feature Pyramid Network (Mamba-BFPN), which incorporates state space modeling and a bidirectional fusion structure to achieve efficient global contextual modeling across feature maps and improve multi-scale information interaction. During multi-scale feature fusion for remote sensing targets, uneven scale perception often arises. Therefore, we design the Multi-Scale Collaborative Guidance Module (MCGM), in which the guidance attention mechanism and semantic collaboration mechanism dynamically adjust the importance of features at different scales, thereby effectively enhancing the model’s perception of dense targets. Finally, to reduce computational overhead and improve the real-time performance of the algorithm, we construct the Structurally Reconfigurable Aggregation Attention (SRAA) module, which combines the local modeling capability of convolution with the global representation advantage of lightweight self-attention, effectively capturing long-range dependencies and improving feature discriminability. Experimental results demonstrate that MambaFNet achieves superior detection accuracy and inference speed on multiple remote sensing datasets, including MAR20 and RSOD. On the MAR20 dataset, MambaFNet attains an AP of 71.1% and a mAP of 97.25%, with 25.91 M parameters and an inference speed of 84.7 FPS, demonstrating strong real-time performance and practical applicability.
